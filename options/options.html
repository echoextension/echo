个性化

ECHO NTP

使用 ECHO 自定义的 NTP

替代默认 NTP，支持鼠标手势和 F2/F3 切换标签

💡 Edge 默认的 NTP 是浏览器内置页面，扩展无法在其上监听鼠标事件。使用 ECHO NTP 后，你可以使用鼠标手势切换标签，且切换时不会跳过 NTP。

此功能由浏览器 API 控制，无法通过开关动态切换。如需使用其他 NTP 扩展，请在 Edge 扩展管理中禁用 ECHO 的 NTP 覆盖。

超级 Bing 悬浮搜索框

启用悬浮搜索框

按下 Ctrl+B 在页面底部呼出搜索框，输入内容后回车进行 Bing 搜索

⚠️ 此功能通过页面注入实现，开启后需刷新页面生效。在输入框中不会拦截，以免影响加粗等操作。在 bing.com 页面自动隐藏。

显示热搜推荐

在搜索框右侧显示热搜推荐词，热词自动滚动轮播，点击直接搜索

默认常驻显示

开启后搜索框在页面加载时自动显示；关闭则需按 Ctrl+B

跟随页面缩放

关闭后搜索框保持固定大小，不随页面缩放变化（默认）；开启后跟随页面缩放

B站视频优化工具

颜色变换、旋转镜像、超级变速

📺
在 Bilibili 视频播放页左侧悬浮显示优化工具
  • 颜色反转与通道交换（全色以及分通道调整）
  • 旋转镜像（以及缩放、适配，特别适合看小姐姐，避免歪脖子）
  • 超级变速（0.25× 超慢速和 3.0× 极速播放）
完全自由组合，也可以随时关闭或暂时隐藏

备份与恢复

备份与恢复

将新标签页壁纸收藏和 ECHO 全部设置导出为文件，防止卸载扩展时数据丢失。重装后可导入恢复。

⚠️ 由于卸载扩展会清除所有数据（包括云端同步数据），建议定期导出备份。所有备份文件仅包含 ECHO 插件版本号、导出时间戳、功能开关设置和新标签页即 NTP 的壁纸收藏。导出逻辑以及保存的 JSON 文件绝对不包含您的任何个人信息、浏览记录或收藏夹数据等。
